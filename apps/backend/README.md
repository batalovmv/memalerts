# MemAlerts Backend

Express API с Socket.IO для Channel Points Mem Alerts.

## Установка

\\\ash
pnpm install
\\\

## Настройка

Создайте \.env\ файл (см. \.env.example\)

## Запуск

\\\ash
# Development
pnpm dev

# Production
pnpm build
pnpm start
\\\

## База данных

\\\ash
pnpm db:push
pnpm db:migrate
\\\
