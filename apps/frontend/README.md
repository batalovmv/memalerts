# MemAlerts Frontend

React приложения для Channel Points Mem Alerts:
- **Web** - основной UI для зрителей и стримеров
- **Overlay** - легкий overlay для OBS Browser Source

## Установка

\\\ash
pnpm install
\\\

## Запуск

\\\ash
# Все приложения
pnpm dev

# Только Web
pnpm dev:web

# Только Overlay
pnpm dev:overlay
\\\

## Сборка

\\\ash
pnpm build
\\\
